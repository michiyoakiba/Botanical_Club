# Botanical Club

## サイト概要
### サイトテーマ
観葉植物の良さを共有するSNSサイト

### テーマを選んだ理由
観葉植物には、種類が豊富で癒し効果があります。素敵な観葉植物を共有したいとの思いから、このテーマにしました。

### ターゲットユーザ
観葉植物が好きな人や興味のある人

### 主な利用シーン
推しの観葉植物を共有したい時、自分に合う観葉植物を探している時
